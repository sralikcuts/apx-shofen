# Arztpraxis Stadelhofen

Statische Website der Arztpraxis Stadelhofen, Goethestrasse 22, 8001 Zürich.

Gebaut mit [Astro](https://astro.build/). Siehe [AGENTS.md](./AGENTS.md) für die Architektur-Notizen.

## Entwicklung

```bash
npm install     # einmalig
npm run dev     # Dev-Server auf http://localhost:4321
npm run build   # Produktions-Build nach ./dist/
npm run preview # Build lokal anschauen
```
